Explanation of each column in the AD results.slsx file:
Type: The type of biologically active RNA node, including: miRNA, circRNA, lncRNA, gene
Symbol: Biologically active RNA node
Score: The probability score that our model predicts about the association
Disease: a disease node
In database: Whether it is in our data
Publication count in database: The number of documents in the data we collect
chromsome: located on that chromosome
Start: The start site of the gene
End: The end site of the gene
